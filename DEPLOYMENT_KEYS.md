### Wiring (website form → n8n → client systems)

```
[ flowlayerai.com form ]
          │  POST JSON
          ▼
[ n8n Webhook (POST) ]
          │
          ▼
[ AI Parser / Qualifier ]
          │
          ├──▶ [ Client Google Sheet ]   (sheet ID = {{CLIENT_DATA_PLACEHOLDER}})
          ├──▶ [ Client Notification  ]   (email / Telegram = {{CLIENT_DATA_PLACEHOLDER}})
          └──▶ [ Webhook Response → form ]
```
